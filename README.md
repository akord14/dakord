# akord.al — skeleton
Hapi 1: Skeleton me rrugë që funksionojnë dhe logo-n zyrtare.

## Komanda
npm install
npm run dev

## Rrugët
- /              (faqja me 2 karta + butona funksionalë)
- /seeking       (placeholder list)
- /offering      (placeholder list)
- /post/new      (form skeleton, pa DB)
- /admin/moderation (placeholder)
- /cv/builder    (KRIJO CV placeholder)

## Konfigurime
- Tailwind është aktiv.
- Vendos variablat Supabase në .env (për hapat në vijim).
